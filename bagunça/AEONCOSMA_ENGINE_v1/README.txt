AEONCOSMA Engine - Pacote Local

Este pacote contém:
- aeoncosma_engine.py: Motor do AEON, com lógica de genoma simbólico, mutações, consciência (CL) e complexidade (K)
- aeon_interface.py: Interface para rodar ciclos no terminal
- aeon_state.json: Arquivo de estado persistente (gerado após execução)

Uso:
1. Execute `aeon_interface.py` com Python 3:
   python aeon_interface.py

2. O sistema irá rodar 13 ciclos e imprimir os resultados.

3. Os dados são salvos automaticamente no arquivo aeon_state.json
